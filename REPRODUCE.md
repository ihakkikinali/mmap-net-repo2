# Reproducing released checks and figures

Install dependencies:

```bash
pip install -r requirements.txt
```

Run consistency checks:

```bash
python scripts/verify_consistency.py
```

Regenerate quantitative figures:

```bash
python scripts/generate_figures.py
```

Expected generated figure stems:

- `fig_03_roc`
- `fig_04_calibration`
- `fig_10_per_label_diff`
- `fig_ablation`
- `fig_deferral_external_device`
- `fig_07_missingness`
- `fig_06_subgroup`
- `fig_domain_shift`
- `fig_09_seed_variability`

The workflow, architecture, and Grad-CAM figures are included but not regenerated by this script.
