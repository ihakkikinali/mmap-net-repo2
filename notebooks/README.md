# Notebooks

The training and evaluation notebooks (data preparation, Protocol-1 T=20
evaluation, Protocol-2 ablation, calibration and deferral) were developed in a
Kaggle GPU environment. They will be released in the public archive upon
acceptance, together with the trained weights.

This review package provides the released predictions, the metric/figure
generation scripts, and the calibration threshold artifacts needed to
reproduce the reported analyses without re-running training.
